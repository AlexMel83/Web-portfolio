CryptoC.exe
Індивідуальний проект по модулю «Алгоритмізація та програмування» для Вінницької ІТ Академії. Програма для шифрування текстових повідомлень афінними та афінними-рекурентними шифрами. Написано на C++ з використанням середовища Windows Forms. Зібрана програма важить лише 149 Кб і не має жодних додаткових залежностей чи бібліотек. Посилання на GitHub: https://github.com/AlexMel83/CryptoC.

Гра "Хрестики-Ноліки" працює на основі JavaScript
Гра "Хрестики-ноліки" написана на javascript. Гра проводиться на сітці 3 квадрати на 3 квадрати. Ти "X", твій друг — "0". Гравці по черзі ставлять свої позначки в порожні квадрати. Гравець, який першим отримає 3 бали, є переможцем. Коли всі 9 квадратів заповнені, гра закінчується.
